Spring Boot MVC Lab

User Management CRUD Practice

A comprehensive hands-on practice project simulating end-to-end data flow in Spring MVC architecture utilizing Thymeleaf Template Engine.

MOCK

Data State: In-Memory Data Store

The application executes core CRUD logic via the Service layer combined with Java Collections and Stream API, isolating the presentation layer before integrating persistent databases.

Functional Core Modules

01 / READ

Fetch Records

Loads data from services, forwards it via Spring Model, and renders elements dynamically using Thymeleaf attributes.

02 / CREATE

Add New User

Leverages Spring's automatic Data Binding engine to handle form submissions securely using @ModelAttribute.

03 / UPDATE

Modify Information

Extracts target identifiers precisely using URL path segments via @PathVariable to mutate existing state.

04 / DELETE

Remove Record

Safely handles state deletion processes, safely orchestrating user re-routing and request lifecycle flows.
